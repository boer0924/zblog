### 环境配置
```bash
git clone <the-git-repo>
cd <the_folder>
npm install
hexo clean
hexo s
hexo g
hexo new post <name-of-post>
hexo new page <name-of-page>
# edit the post
git add <name-of-post>
git commit -m <message-of-commit>
git push
# travis-ci
```
